The maker

munax

Munavvir is a mobile technician based in Kerala who built UNIPARTS to solve a real daily problem — finding compatible universal parts without relying on memory or WhatsApp groups.

Under the munax brand, this database was engineered entirely from a phone. No PC, no team — just code and a genuine need to build something useful for the repair community.

From scraping to search engine to UI — every layer built with precision. This is the result.
